광고 → WhatsApp 즉시 연결

오픈율 98%
2
방문 일정 수집 → 트리거 기준 설정
입력된 방문 날짜는 Phase 2 도착 트리거의 기준이 됩니다. 별도 액션 없이 방문일 당일 오전 10시에 자동 메시지가 발송됩니다.
챗봇 완료율 60~75%
3
숙박 지역 → 맞춤 매장 안내
홍대·성수 선택 시 무신사 스탠다드 직영점 정보 자동 강조. 출발 3일 전 숙소에서 가장 가까운 매장 정보와 영업시간이 자동 발송됩니다.
매장 방문 의향 +40%
4
회원가입 → 15% 쿠폰 즉시 발송
챗봇에서 바로 무신사 회원가입 링크를 제공하고, 가입 완료 즉시 방문 기간에 사용 가능한 15% 할인 쿠폰을 WhatsApp으로 자동 발송합니다.
쿠폰 사용률 40~60%
PHASE 1 → PHASE 2

한국 도착 — 트리거 발송 & 핫 아이템 안내

도착 확인 → 핫 아이템 즉시 발송
"도착했어요!" 응답 즉시 오늘의 핫 아이템 TOP 3와 매장 지도 링크가 자동 발송됩니다. 홍대 지역 선택자에게는 무신사 스탠다드 홍대점 정보가 우선 노출.
매장 방문 전환율 25~40%
3
사전 발급 쿠폰 활성화 안내
Phase 1에서 발급받은 15% 쿠폰이 도착일부터 활성화됩니다. 계산대에서 이 WhatsApp 화면을 보여주면 즉시 적용되는 방식으로 마찰을 최소화합니다.
쿠폰 사용률 40~60%
4
체류 기간 일별 Flash Deal 발송
방문 기간 동안 매일 오전 현지 시간 기준으로 오늘의 Flash Deal과 신상 입고 알림이 자동 발송됩니다. 마지막 날에는 쿠폰 만료 D-day 알림.
일별 메시지 클릭률 45~55%
PHASE 2 → PHASE 3

귀국 후 — 재방한 서베이 & 바이럴 루프
방문 일정 종료일 +1일부터 귀국 시퀀스가 자동 시작됩니다. 서베이 응답 기반으로 재방한 유도 쿠폰이 발송되고, 지인 추천으로 바이럴 루프가 완성됩니다.

재방한 예약 의향 25~35%

지인 추천 전환율 8~15%
